# Product Demo Content

Copy `src/config/product-demo.ts` into the separate Karpilo LoadIQ website repo at:

```text
src/config/product-demo.ts
```

Use it to render:

- an interactive-looking demo card
- a sample LoadIQ result panel
- an estimate vs actual comparison block
- a "How LoadIQ Thinks" step sequence
- a CTA below the demo

The content is static, fictional, and non-sensitive. It does not connect to Supabase, does not require authentication, and should not be represented as a live freight quote or guaranteed profitability outcome.
